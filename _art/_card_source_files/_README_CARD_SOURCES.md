The ability and character cards were created using Penpot (https://penpot.app/).

I actually think I came up with a pretty cool template to minimize the amount of work needed to create new cards or edit old ones, feel free to check it out and reuse it for your purposes as desired!

These cards used various artwork shamelessly pulled from screenshots of the movie, so obviously please don’t use this for commercial purposes.  Unless this somehow gets licensed as an official Huntr/x x Hero Quest crossover product.  That would be sick.

The backside of the ability cards were created using paint.net, you should be able to open the pdn files with that app if you want to modify them further.
