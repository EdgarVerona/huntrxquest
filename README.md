# Huntr/x Quest: A HeroQuest Conversion

![I wish you could see this jank-ass logo](https://github.com/EdgarVerona/huntrxquest/blob/main/_art/huntrx-quest-logo.jpg)

*The unauthorized. unasked-for, and quite frankly ludicrous K-Pop Demon Hunters x Hero Quest conversion*

Version 0.1, still undergoing testing but should be in an "alpha playable" state.  I welcome any playtesting you'd be willing to undertake with your girl squad!

## What IS this?

Huntr/x Quest is a Hero Quest mod that provides:

- 3 new Hero characters, styled to the heroes of the K-Pop Demon Hunters movie.
- A new system for character growth through upgrades and abilities rather than items.
- Many new required/optional changes, both to fit the theme and to streamline gameplay.
- An 11 Quest campaign that takes the girls through their first world-saving mission, shortly after forming Huntr/x.
- Lots of fun surprises, with maybe too many board overlays to allow for said surprises.

The files in this project contain the assets and documents needed to play or modify Huntr/x Quest.

## What do I need to play?

- A physical copy of Hero Quest - either the original 1989/1990 printing, or the 2020+ reprint.

- The printouts suggested below.

- A party of 3 people to play as Huntr/x, or fewer people willing to play multiple characters.

- A person willing to play the role of Gwi-Ma (Zargon in HeroQuest terminology, or the "Dungeon Master" if you're into D&D).

## Files and Folders

Here's more details about what specifically is in each file and folder:

* **huntrx-armory-reference.pdf** \- Contains all the buy and sell information for items in the game, including “core” items, “expansions” if you want to mod this further, and items/enhancements added for purchase by this campaign itself.

* **huntrx-keywords-reference.pdf** \- Contains all of the “keywords” I use in the cards, quests, etc… so that I don’t have to describe the same concept multiple times.  Good reference for Gwi-Ma as well as players.

* **huntrx-rules-addendum.pdf** \- Contains all of the new and changed rules (optional and required) to Hero Quest for running the campaign.  Players are encouraged to read this.

* **player-references/player-reference-card.png** \- This is a quick reference card for players to see what they can do every turn.

* **FOLDER: gwi-ma-resources**  
  * **agwi-hwasin-phase-guide.pdf** \- A summary of the main villain’s abilities and how to run him on the table, as well as optional rules for if you want to run him less like a predictable “video game” style boss and more like a human-driven intelligence.  ***Contains major spoilers, do not show this or even mention its existence to players\!***

  * **gwi-ma-rules-addendum.pdf** \- A summary of the new rules \- optional as well as required \- for the person playing Gwi-Ma/Zargon.  **Contains major spoilers, do not show this to players\!**

  * **huntrx-monster-chart.pdf** \- A summary of the stats for each non-boss monster in the game.  Gwi-Ma should print this out and keep it for their reference, but don’t show it to players.

  * **FOLDER: board-overlays** 
    * **FOLDER: dungeondraft-source-maps** \- Useful if you want to modify any of these overlays.  
    * **1989/1990** \- Open/Print these if you have the original version of the board, which has \~22.5mm wide squares.  
      * **FOLDER: a4-print-ready / letter-print-ready** \- Easiest way to print and play the board overlays.  Read the readme file inside for more info.  
      * **FOLDER: full-res-pngs** \- If you instead want to print the overlays on a poster or some other large format printer, these are the original source overlays as unsplit and unmultiplied images.  I leave the combination of these into large format printing as an exercise for the reader.  
    * **2020** \- Open/Print these if you have the 2020 remake of the game, and thus the board with larger 1 inch squares.  
      * **FOLDER: a4-print-ready / letter-print-ready** \- Easiest way to print and play the board overlays.  Read the readme file inside for more info.  
      * **FOLDER: full-res-pngs** \- If you instead want to print the overlays on a poster or some other large format printer, these are the original source overlays as unsplit and unmultiplied images.  I leave the combination of these into large format printing as an exercise for the reader.

  * **FOLDER: quests**  
    * **\# (Quest Name) PDF files** \- The exported PDFs from hquestbuilder for each quest in the campaign.  Note that the final quest, \#11, has 3 quest files split into phases.  There’s a reason for that which will be obvious when you read them. Play them in order, it will tell you what to do when the time comes. **Contains major spoilers, do not show this or even mention its existence to players\!**

    * **hquestbuilder-quest-backup.hqb** \- The hquestbuilder backup file for the source of the quests, in case you want to modify them further.  **Contains major spoilers, do not show this or even mention its existence to players\!**

* **FOLDER: character-cards**
  * Print all of these png files to print out the character cards.

* **FOLDER: ability-cards**
  * Print all of these png files to print out the ability cards.  You can also choose to print every possible copy of an ability card in advance, or wait until someone actually chooses to have multiple of a given ability before printing them.

* **FOLDER: art**  
  * If you want to use penpot.app to make your own cards or modify the ones here, you can import these files.  Or if you want the jank logo I slapped together from the Hero Quest box art combined with the K-Pop movie art for some reason.
